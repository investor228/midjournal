# 📖 AI MoodJournal (ИИ-Дневник)

Полноценное Fullstack-приложение для ведения личного дневника. Особенность проекта в том, что на каждую вашу мысль или запись интегрированная нейросеть (Google Gemini) дает короткий, поддерживающий и ободряющий ответ.

## ✨ Главные фишки (Features)
* **CRUD операции:** Создание, чтение и удаление записей.
* **Интеграция с ИИ:** Генерация ответов с помощью Google Gemini API.
* **Умный поиск:** Мгновенная фильтрация истории записей (без перезагрузки страницы).
* **Горячие клавиши:** Отправка мыслей по нажатию `Enter` (и перенос строки через `Shift + Enter`).
* **Современный UI:** Адаптивный дизайн, плавные анимации и мягкие тени (Vanilla CSS).

## 🛠️ Стек технологий
**Фронтенд:**
* HTML5, CSS3, Vanilla JavaScript
* Сборщик: Vite

**Бэкенд:**
* Node.js & Express
* База данных: PostgreSQL (в контейнере Docker)
* ORM: Prisma
* ИИ: Google Generative AI SDK (`gemini-2.5-flash`)
