# 📖 AI MoodJournal — Твой умный ИИ-Дневник

[🌐 **Посмотреть проект вживую (Live Demo)**](https://midjournal-one.vercel.app/)

Полноценное Fullstack-приложение (SaaS) для ведения личного дневника. Особенность проекта — интеграция с передовой нейросетью Google Gemini, которая анализирует ваши записи и дает короткие, ободряющие и поддерживающие ответы.

## ✨ Главные фишки
* **Умный ИИ-собеседник:** Интеграция с Google Gemini API для генерации ответов.
* **Система аккаунтов (Auth):** Регистрация, логин и безопасное хеширование паролей.
* **Приватность и безопасность:** Защита маршрутов с помощью JWT-токенов (каждый пользователь видит и управляет только своими записями).
* **Кастомизация UI:** Поддержка светлой и темной (Dark Mode) темы с сохранением выбора пользователя в LocalStorage.
* **Умный поиск:** Мгновенная "живая" фильтрация мыслей и ответов ИИ без обращений к серверу.
* **Настройки профиля:** Возможность менять имя, email и безопасно обновлять пароль.

## 🛠 Стек технологий
* **Фронтенд:** HTML5, CSS3 (кастомные переменные), Vanilla JavaScript, сборщик Vite.
* **Бэкенд:** Node.js, Express.js.
* **База данных:** PostgreSQL (развернута в облаке Neon.tech), ORM Prisma.
* **Безопасность:** `jsonwebtoken` (JWT), `bcrypt`.
* **Деплой (Хостинг):** Vercel (Фронтенд), Render (Бэкенд).

## 📂 Структура проекта
* `/frontend` — клиентская часть (UI, стили, запросы к API).
* `/backend` — серверная часть (REST API, работа с базой данных, ИИ, авторизация).

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/ВАШ_НИК/midjournal.git](https://github.com/ВАШ_НИК/midjournal.git)
2. Настройте базу данных в облаке (например, Neon.tech) и получите ключи от Google Gemini AI.

2. Создайте файл .env в папке /backend и добавьте переменные:

Code snippet
DATABASE_URL="ваша_ссылка_на_postgres"
GEMINI_API_KEY="ваш_ключ_gemini"
JWT_SECRET="ваш_секретный_код"
4. Установите зависимости и запустите бэкенд:

Bash
cd backend
npm install
npx prisma migrate dev
node server.js
5. В новом терминале запустите фронтенд:

Bash
cd frontend
npm install
npm run dev
*(Не забудь поменять `ВАШ_НИК` в ссылке на клонирование на свой реальный ник в GitHub).*
